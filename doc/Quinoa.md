
# Quinoa
Quinoa 是一款专为知识管理打造的现代化 Hexo 主题，全端适配、极速加载、简单易用，让你的知识库随时随地触手可及。

## 📚 hexo-theme-Quinoa 项目特点
### 🌐 全端支持，随时随地访问
- 响应式设计：完美适配桌面、平板、手机等多种设备，无论在大屏还是小屏上都能获得优质阅读体验
- 移动优先：专为移动端优化的可折叠侧边栏、触摸友好的交互界面，手机上也能轻松导航
- 自适应布局：根据不同屏幕尺寸自动调整内容展示，确保最佳的可读性

### ⚡ 极速加载，流畅体验
- SPA 全站模式：基于 Hash 的单页应用导航，页面切换无刷新，极速响应
- 局部加载优化：文章加载从全局刷新改为仅加载内容，首屏渲染时间减少 40%
- 静态站点生成：纯静态 HTML 输出，无需数据库，部署简单，访问飞快
- 客户端搜索：基于 Insight Search 的闪电般全文检索，无需后端支持

### 🎯 简单易用，专注内容
- Markdown 原生：专注写作，无需学习复杂语法，完美支持 AI 内容生成
- 层级分类管理：侧边栏多级目录树，支持展开/折叠，知识组织清晰有序
- 双模式路由：全站 Wiki 体验与单页分享模式无缝切换，满足不同场景需求
- 一键部署：基于 Hexo 生态，Git 工作流支持，版本控制与内容管理一体化

### 🔧 功能丰富，扩展性强
- 全文搜索：内置客户端搜索，快速定位内容
- 代码高亮：多主题语法高亮，技术文档展示更专业
- 插件生态：支持 MathJax、Mermaid 图表、音乐播放器、文章加密等扩展
- SEO 优化：自动生成站点地图、RSS 订阅，搜索引擎友好
